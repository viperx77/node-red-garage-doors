Garage-Doors
============

#### Features:

1. Open and Close Doors from dashboard buttons
2. Supports sensors for button colors and transitioning
3. Closes doors at specified time if open
4. emails if open/close transition does not complete

Button Colors:
- RED = Open
- GREEN = Closed
- BLUE = Transitioning
